jQuery Slider 7

Install slider

You can install using bower or you can download zip file from github. See the source in the github page.

$ bower install slider7

Add javascript code in your html 

<script type="text/javascript" src="/js/file/path/slider7.min.js"></script>

Image list, with main container and img tags

<div class="slider7">
    <div>
        <a href="#"><img src="photo1.jpeg"></a>
    </div>
    <div>
        <a href="#"><img src="photo2.jpeg"></a>
    </div>

        <a href="#"><img src="photo4.jpeg"></a>
    </div>
    <div>
        <a href="#"><img src="photo5.jpeg"></a>
    </div>
</div>

Load plugin on the html

jQuery(window).load(function(){
    jQuery( ".slider7" ).slider7();
});
Use jQuery(window).load() instead of jQuery(document).ready() because the plugin needs image information like height and width to build itself.

Example

All the options avalaible

jQuery( ".slider7" ).slider7({
    height: "400px",
    imgContent: "div",
    imgMarginRight: "15px",
    speed: 15
});
